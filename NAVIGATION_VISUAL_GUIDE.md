### Behavior Changes

**Mobile (< 768px)**
- Show hamburger menu
- Hide desktop navigation
- Stack footer columns
- Full-width dropdowns

- Show desktop navigation
- Smaller dropdown (600px)
- 2-column footer
- Condensed spacing

**Desktop (> 1024px)**
- Full navigation
- Large dropdown (800px)
- 4-column footer
- Optimal spacing

**Large (> 1280px)**
- Extra large dropdown
- More whitespace
- Enhanced hover effects

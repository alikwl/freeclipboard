## Content Guidelines

### Writing Style
- Clear, concise, and helpful
- Use bullet points and lists
- Include examples where relevant
- Use emojis sparingly for visual interest
- Write for beginners but include advanced tips

### SEO Optimization
- Use keywords naturally in headings
- Include long-tail keywords in FAQ
- Add schema markup for FAQs
- Link to related tools internally
- Use descriptive anchor text

### User Experience
- Progressive disclosure (basic ‚Üí advanced)
- Scannable content with clear headings
- Visual hierarchy with proper HTML tags
- Mobile-friendly formatting
- Accessible markup (ARIA labels, semantic HTML)

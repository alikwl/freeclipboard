### Optional Enhancements

#### 1. Add More Free Resources
When adding new resources:
```html
<!-- In free-resources.html -->
<div class="resource-card">
  <a href="{{ site.baseurl }}/free-resources/[new-resource]/">
    <!-- Resource content -->
  </a>
</div>
```

```xml
<!-- In sitemap.xml -->
<url>
  <loc>{{ site.url }}/free-resources/[new-resource]/</loc>
  <lastmod>{{ site.time | date_to_xmlschema }}</lastmod>
  <changefreq>monthly</changefreq>
  <priority>0.8</priority>
</url>
```

#### 2. Add Breadcrumb Navigation
Consider adding breadcrumbs to resource pages:
```
Home > Free Resources > Flow Veo Automator
```

#### 3. Add Related Resources Section
On individual resource pages, link to related resources:
```html
<section class="related-resources">
  <h2>Related Resources</h2>
  <a href="/free-resources/[other-resource]/">...</a>
</section>
```

#### 4. Add Back to Hub Link
On resource detail pages, add prominent link back to hub:
```html
<a href="{{ site.baseurl }}/free-resources/" class="back-link">
  ← Back to Free Resources
</a>
```

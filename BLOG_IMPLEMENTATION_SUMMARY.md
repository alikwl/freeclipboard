## üìù Content Guidelines for Future Posts

When creating new blog posts, follow this structure:

1. **Front Matter**
   - Layout: post
   - Title (60 characters max)
   - Description (155 characters max)
   - Date
   - Author
   - Category
   - Tags (5-7 tags)
   - Keywords
   - Image path
   - Featured (true/false)
   - Read time
   - Related tools

2. **Content Structure**
   - Introduction (hook + value proposition)
   - Table of contents (auto-generated)
   - Main sections (H2 headings)
   - Subsections (H3 headings)
   - Lists and tables
   - Images and visuals
   - CTA boxes (2-3 per post)
   - Conclusion
   - Related resources

3. **SEO Elements**
   - Primary keyword in title
   - Keywords in headings
   - Internal links (5-10)
   - External links (3-5)
   - Alt text for images
   - Meta description
